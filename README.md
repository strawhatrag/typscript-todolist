# TypeScript TodoList

A simple TodoList web app built using TypeScript, HTML, and CSS. Users can add, remove, and mark tasks as completed. Data is saved locally using browser's localStorage.

## Features

- Add, mark, and remove tasks.
- Data persistence using localStorage.

## Getting Started

1. Clone the repo or download files.
2. Install dependencies: `npm install`.
3. Run the app: `npm start`.

## Usage

1. Enter a task, click "Add" to add it.
2. Click checkbox to mark as completed.
3. Click "Remove" to delete a task.
4. Data is saved and persists across page reloads.
